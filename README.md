**bel_adn** is the common library for the frontend of the "magnifique couple" project. 

## Features
The library contains the binding for the api of "magnifique couple"

## Getting started

Just add the library to your deps :
```yml
    bel_adn: 0.0.1
```

and :
```bash
    dart pub run update
```
